# Folder Encryptor V5

`Folder Encryptor V5` は、Windows 向けの軽量なフォルダー暗号化ツールです。

このリポジトリでは、すぐに使える配布パッケージとドキュメントを公開しています。ダウンロード、体験、共有をしやすくすることが目的です。ソースコード、ライセンスの詳細、より完全な技術説明は今後も順次整理していきます。

## ソフトウェア概要

- Windows デスクトップ向けのフォルダー暗号化・復号ツール
- インストーラー版とポータブル版を提供
- 手動パスワード入力と強力なランダムパスワード生成に対応
- ランダムパスワード生成時、デスクトップ上のテキストファイルへパスワード記録を追記保存

## 主な機能

- 対象フォルダーを選んで暗号化または復号
- 任意のマスターパスワードを入力可能
- `随机生成` ボタンで強力なパスワードを即時生成
- ランダムパスワードをデスクトップへ自動バックアップ
- 複数回生成しても既存内容を上書きせず、同じバックアップファイルへ追記

## ランダムパスワードとバックアップ

自分でパスワードを設定したくない場合は、アプリ内の `随机生成` ボタンをクリックしてください。

プログラムは次の処理を行います。

1. 強力なパスワードを自動生成
2. 生成したパスワードをデスクトップ上のバックアップ用テキストファイルに保存
3. 次回以降も同じファイルに追記し、以前の内容は上書きしない

例:

```text
【V5 专属加密密码备份】
密码: euZi^oahxuKq2Zm6

【V5 专属加密密码备份】
密码: JSE7CWTg54#CSH!Q
```

暗号化完了後は、バックアップファイルを安全な場所に保管し、暗号化ファイルと長期間同じ場所に置かないことをおすすめします。

## クイックスタート

### 1. 版を選ぶ

- 通常インストールしたい場合は `FolderEncryptor_Dino_SmartSetup.exe`
- インストール不要で使いたい場合は `FolderEncryptor_V5_Dino.exe`

### 2. フォルダーを暗号化する

推奨手順:

1. `Folder Encryptor V5` を起動
2. `Target Folder / 目标文件夹` で保護したいフォルダーを選択
3. `Master Password / 安全密码` にパスワードを入力するか、`随机生成` をクリック
4. `隐形加密 (Encrypt)` をクリック
5. 処理完了後、結果を確認

### 3. フォルダーを復号する

1. `Folder Encryptor V5` を起動
2. すでに暗号化されたフォルダーを選択
3. 正しいパスワードを入力
4. `一键还原 (Decrypt)` をクリック
5. ファイルが正常に復元されたことを確認

## 利用時のおすすめ

- 重要なデータの前に、まずテスト用フォルダーで一通り試す
- 重要ファイルを扱う前に追加バックアップを作成する
- ランダムパスワードを使う場合は、デスクトップ上のバックアップファイルを必ず保管する
- 現在の公開版は主に Windows 環境向け
