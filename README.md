# FolderEncryptor Dino

`FolderEncryptor Dino` 是一个面向 Windows 的轻量级文件夹加密小工具。

当前仓库先公开发布包、使用文档和发布文章，方便大家直接下载体验；源码、许可证和更完整的技术说明会在后续整理后逐步补充。

## 项目定位

- 面向 Windows 桌面环境的文件夹加密/解密小工具
- 提供安装版和便携版两种分发方式
- 适合个人隐私资料、课程资料、项目文档等文件夹的本地整理与保护场景
- 当前仓库以“可直接使用”为目标，优先提供发布包与说明文档

## 快速开始

### 1. 选择运行方式

- 如果你希望正常安装，优先使用 `FolderEncryptor_Dino_SmartSetup.exe`
- 如果你希望免安装直接体验，可以使用 `FolderEncryptor_V5_Dino.exe`

### 2. 加密文件夹

推荐流程如下：

1. 启动程序
2. 选择需要保护的目标文件夹
3. 输入并确认加密密码
4. 执行加密操作
5. 完成后先检查结果，再决定是否删除原始备份

### 3. 解密文件夹

1. 启动程序
2. 选择已经加密过的文件夹
3. 输入加密时使用的同一密码
4. 执行解密操作
5. 确认文件恢复正常后再继续后续处理

## 使用建议

- 第一次使用时，建议先拿一份测试文件夹演练完整流程
- 重要资料请先做额外备份，再进行加密操作
- 请务必妥善保存密码；如果密码遗失，恢复数据会非常困难
- 当前公开版本主要面向 Windows 使用环境
