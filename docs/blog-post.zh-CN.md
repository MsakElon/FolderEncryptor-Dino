# 我把自己做的一个文件夹加密小工具公开到了 GitHub

最近把之前做的一个小软件重新整理了一遍，决定正式公开出来。

它叫 `FolderEncryptor Dino`，是一个面向 Windows 的轻量级文件夹加密工具。这个项目最早是我在个人需求驱动下做出来的，后来又结合 OpenClaw 的协助，把功能逐步补到了一个“自己真的会长期用”的状态。

这次公开，主要想做两件事：

1. 把它作为一个完整的小作品认真整理出来
2. 让需要类似工具的朋友可以直接下载体验

## 这个工具是做什么的

一句话介绍：

它是一个帮助你对本地文件夹进行加密和解密的小工具。

我做它的初衷其实很简单。平时总会有一些不想随手裸放的资料，比如：

- 课程资料
- 项目文档
- 个人隐私文件
- 临时整理的一组敏感数据

这些内容未必需要上特别重的企业级方案，但又确实希望有一个更顺手、更轻量的本地工具来处理。于是就有了这个项目。

## 如何使用

目前提供两种方式：

### 方式一：安装版

下载并运行 `FolderEncryptor_Dino_SmartSetup.exe`，按正常流程安装后即可使用。

### 方式二：便携版

如果你不想安装，也可以直接运行 `FolderEncryptor_V5_Dino.exe`。

### 基本使用流程

加密文件夹时：

1. 打开程序
2. 选择要处理的文件夹
3. 输入并确认密码
4. 执行加密
5. 完成后检查结果

解密文件夹时：

1. 打开程序
2. 选择已经加密过的文件夹
3. 输入原始密码
4. 执行解密
5. 确认文件恢复正常

第一次使用时，我非常建议先拿一份测试文件夹演练一遍流程，不要直接对唯一原件开刀。
