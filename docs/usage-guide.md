# Usage Guide

## 适用对象

这份说明适合第一次接触 `FolderEncryptor Dino` 的用户。

如果你只是想快速上手，可以先看 README；如果你准备正式使用，建议完整读完本页。

## 运行方式

项目当前提供两种使用方式：

- 安装版：`dist/FolderEncryptor_Dino_SmartSetup.exe`
- 便携版：`dist/FolderEncryptor_V5_Dino.exe`

一般建议优先使用安装版；如果只是临时体验或不想安装，可以直接运行便携版。

## 开始前建议

在第一次使用前，建议先准备一份测试文件夹，里面放几份不重要的文档或图片，先完整跑通一次加密和解密流程。

这样做的好处是：

- 你可以先熟悉界面和操作顺序
- 可以确认当前电脑环境运行正常
- 可以避免在第一次使用时直接处理重要资料

## 文件夹加密

推荐操作步骤：

1. 启动程序
2. 在界面中选择要加密的目标文件夹
3. 输入密码，并再次确认密码
4. 检查目标路径是否正确
5. 执行加密
6. 等待程序处理完成
7. 打开结果目录做一次人工确认

## 文件夹解密

推荐操作步骤：

1. 启动程序
2. 选择已被该工具处理过的文件夹
3. 输入原始加密密码
4. 执行解密
5. 检查恢复后的文件能否正常打开

## 使用中的注意事项

### 1. 密码管理

- 密码是恢复文件的关键
- 请务必自行保存好密码
- 不要只依赖临时记忆

### 2. 先备份再操作

- 对重要资料进行加密前，建议额外备份一次
- 尤其是第一次正式使用时，不建议直接处理唯一原件

### 3. 先小规模测试

- 先拿一两个小文件夹测试最稳妥
- 确认流程稳定后，再处理更重要的数据

### 4. 环境建议

- 当前公开版本面向 Windows 桌面环境
- 如果系统安全策略较严格，首次运行可能需要确认是否允许执行

## 常见问题

### 安装版和便携版有什么区别？

- 安装版更适合作为常用工具长期保留
- 便携版更适合快速体验和单文件分发

### 忘记密码怎么办？

密码是加密数据恢复的重要凭据。建议在开始使用前，就确定好自己的密码管理方式；如果密码遗失，恢复会非常困难。

### 可以直接拿重要资料做第一次测试吗？

不建议。推荐先用测试文件夹验证流程，再处理正式数据。

## 推荐使用场景

- 个人隐私资料整理
- 课程作业和学习资料归档
- 本地项目文档的额外保护
- 需要临时打包保护的一组文件夹
